# Quick Setup - Create Repo and Push

## Option 1: Create via Web (Easiest)

1. **Click this link to create the repository with pre-filled values:**
   https://github.com/new?name=slotly-marketing&description=Marketing+website+for+Slotly+scheduling+platform

2. **Make sure:**
   - Repository name is: `slotly-marketing`
   - **DO NOT** check any boxes (no README, .gitignore, or license)
   - Click **"Create repository"**

3. **After creating, come back and I'll push the code automatically!**

## Option 2: Manual Steps

If the link doesn't work:

1. Go to: https://github.com/new

3. Description: "Marketing website for Slotly scheduling platform"
4. **DO NOT** initialize with README, .gitignore, or license
