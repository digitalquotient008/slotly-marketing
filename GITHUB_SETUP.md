# GitHub Repository Setup

## Create Repository on GitHub

1. Go to https://github.com/new
2. Repository name: `meeteasy-marketing`
3. Description: "Marketing website for MeetEasy scheduling platform"
4. Visibility: Public (or Private if preferred)
5. **DO NOT** initialize with README, .gitignore, or license (we already have these)
6. Click "Create repository"

## Push Code to GitHub

After creating the repository, run:

```bash
cd /Users/girishreddy/DEV_WORKSPACES/meeteasy-marketing
git remote add origin https://github.com/digitalquotient008/meeteasy-marketing.git
git branch -M main
git push -u origin main
```
